API Endpoints

  • Teacher related URLs
    • GET /api/teacher/ - List of teachers
    • GET /api/teacher/dashboard/{id}/ - Teacher dashboard
    • GET /api/teacher/{id}/ - Get teacher details
    • POST /api/teacher/change-password/{id}/ - Change teacher password
    • POST /api/teacher-login/ - Teacher login
    • GET /api/popular-teachers/ - Popular teachers
    • GET /api/verify-teacher/{id}/ - Verify teacher via OTP
    • POST /api/teacher-forgot-password/ - Forgot teacher password
    • POST /api/teacher-change-password/{id}/ - Change teacher password
  • Course related URLs
    • GET /api/category/ - List of categories
    • GET /api/course/ - List of courses
    • GET /api/popular-courses/ - List of popular courses
    • GET /api/search-courses/{searchstring}/ - Search for courses
    • POST /api/update-view/{course_id}/ - Update course view count
    • GET /api/course/{id}/ - Get course details
    • GET /api/chapter/{id}/ - Get chapter details
    • GET /api/course-chapters/{course_id}/ - List of chapters for a course
    • GET /api/teacher-courses/{teacher_id}/ - List of courses by a teacher
    • GET /api/teacher-course-detail/{id}/ - Get course details for a teacher
  • Student related URLs
    • GET /api/student-testimonial/ - List of student testimonials
    • GET /api/student/ - List of students
    • GET /api/student/dashboard/{id}/ - Student dashboard
    • GET /api/student/{id}/ - Student details
    • POST /api/student-login/ - Student login
    • POST /api/student/change-password/{id}/ - Change student password
    • GET /api/student-enroll-course/ - List of enrolled courses for a student
    • GET /api/fetch-enroll-status/{student_id}/{course_id}/ - Fetch enrollment status
    • GET /api/fetch-all-enrolled-students/{teacher_id}/ - List of all enrolled students for a teacher
    • GET /api/fetch-enrolled-students/{course_id}/ - List of enrolled students for a course
    • GET /api/fetch-enrolled-courses/{student_id}/ - List of enrolled courses for a student
    • GET /api/fetch-recommended-courses/{studentId}/ - List of recommended courses for a student
    • GET /api/course-rating/{course_id}/ - Rating for a course
    • GET /api/fetch-rating-status/{student_id}/{course_id}/ - Fetch rating status for a student
    • GET /api/student-add-favourite-course/ - Add course to favorites for a student
    • DELETE /api/student-remove-favourite-course/{course_id}/{student_id}/ - Remove course from favorites for a student
    • GET /api/fetch-favourite-status/{student_id}/{course_id}/ - Fetch favorite status for a course
    • GET /api/fetch-favourite-courses/{student_id}/ - List of favorite courses for a student
    • GET /api/student-assignment/{teacher_id}/{student_id}/ - List of assignments for a student
    • GET /api/my-assignments/{studentId}/ - List of assignments for a student
    • PUT /api/update-assignment/{id}/ - Update an assignment
    • GET /api/student/fetch-all-notifications/{studentId}/ - List of all notifications for a student
    • POST /api/save-notifications/ - Save notifications for a student
  • Quiz related URLs
    • GET /api/quiz/ - List of quizzes
    • GET /api/teacher-quiz/{teacher_id}/ - List of quizzes created by a teacher
    • GET /api/teacher-quiz-detail/{id}/ - Details of a quiz created by a teacher
    • GET /api/quiz/{id}/ - Details of a quiz
    • GET /api/quiz-questions/{quiz_id}/ - List of questions for a quiz
    • GET /api/quiz-questions/{quiz_id}/{limit}/ - List of limited questions for a quiz
    • GET /api/fetch-quiz-assign-status/{quiz_id}/{course_id}/ - Fetch quiz assignment status
    • POST /api/quiz-assign-course/ - Assign quiz to a course
    • GET /api/fetch-assigned-quiz/{course_id}/ - List of quizzes assigned to a course
    • POST /api/attempt-quiz/ - Attempt a quiz
    • GET /api/quiz-questions/{quiz_id}/next-question/{question_id}/ - Next question in a quiz
    • GET /api/fetch-quiz-attempt-status/{quiz_id}/{student_id}/ - Fetch quiz attempt status
  • Study materials related URLs
    • GET /api/study-materials/{course_id}/ - List of study materials for a course
    • GET /api/study-material/{id}/ - Details of a specific study material
    • GET /api/student/study-materials/{course_id}/ - List of study materials for a course (student view)
    • GET /api/attempted-quiz/{quiz_id}/ - List of attempted quizzes
    • GET /api/fetch-quiz-result/{quiz_id}/{student_id}/ - Fetch result of a quiz for a student
  • FAQ and contact URLs
    • GET /api/faq/ - List of FAQs
    • GET /api/contact/ - Contact information

Usage

Here's how you can use the Virtu Learn Backend API:

  1. Make HTTP requests to the appropriate endpoint using tools like Postman or cURL.
  2. Include necessary parameters and headers in your requests.
  3. Handle the responses based on the provided status codes and response data.
